About this Website

CODEY is an online coding platform designed to be the central hub for our college coding fest. Its core functionality is to provide a seamless and interactive coding experience for participants. The platform's key features include:

  • **Real-time Code Execution:** Powered by the Judge0 API, the platform allows you to compile and run code in over 7 different programming languages instantly.
  • **Interactive Terminal:** The output terminal is not just for displaying results; it also supports user input, making it possible to test programs that require dynamic parameters.
  • **Animated & Responsive UI:** The user interface is built with a focus on animation and a cool, modern design that is fully responsive on both desktop and mobile devices.
  • **Code Persistence:** Your last two code submissions are securely stored in a SQL database, allowing you to easily revisit and continue working on your previous attempts.

This website is an embodiment of modern web development principles, combining a powerful backend with a stunning and intuitive frontend.
